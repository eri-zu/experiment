# Fullscreen Animation

## 📝 Overview

A fullscreen animation demo built with Three.js and glsl.

## 🧠Study Note

- fullscreen におさまるように scale を計算
- 同時に画面中心に向かって動かす
- 座標に応じてアニメーションの開始タイミングをずらす
